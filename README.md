# distribox
Sistema de análise estratégica para varejo de moda com 15 lojas. Consolida dados de vendas, produtos, clientes e estoque em dashboard interativo. Permite consulta de produtos com estoque, análise de vendas com filtros, projeção por regressão linear e geração de relatório executivo para diretoria (RAE). Interface Streamlit com gráficos Plotly.
